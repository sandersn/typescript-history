How to Contribute to TypeScript

If you would like to contribute, consider these options:
  • Contribute bug fixes. This includes information on the Contributor License Agreement (or CLA)
  • Discuss the language specification (docx, pdf) in the TypeScript discussion forum.
  • Review code checkins and send feedback. (The develop branch has the latest changes.)
  • Submit a bug report (for a guide on submitting good bug reports, read Painless Bug Tracking).
  • Help answer questions on StackOverflow.
  • Join the #typescript discussion on Twitter.
  • Tell others about the project.
